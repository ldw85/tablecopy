About this tool (How it works & EEAT)

实现与维护：该工具页面由 TableCopy 团队维护。核心截取逻辑使用社区主流库 html2canvas，并在脚本中实现了加载失败提示与回退机制以提高健壮性。

技术细节：脚本在注入时会临时隐藏自身生成的模态（避免被截图），并通过 `useCORS` 提升跨域图片资源的渲染成功率。图片导出采用 dataURL 的方式触发浏览器下载。

可信性与可审计性：项目为静态代码库，所有逻辑在 tools/tablecopy/ 可查看；第三方依赖从 CDN 加载，若需离线部署可将脚本打包托管于可信源。

FAQ

What if overlays or toolbars appear in the capture?

The script tries to hide its own modal and controls to avoid self-capture. For third-party overlays (chat widgets, cookie notices or banners), please close or hide them before capturing.

Why are some images or resources missing in the exported image?

If resources come from third-party domains without CORS headers, the browser can block drawing them to the canvas. Open in a new tab or save critical images locally before composing a capture as a workaround.

How to get higher resolution or smaller file size?

Increase the scale (e.g. window.devicePixelRatio * 2) for higher resolution, but this uses more memory. To reduce file size, lower quality parameters or compress the file afterwards.

How to avoid memory crashes on very long pages?

Very long pages can use lots of memory. Consider capturing in segments or lowering scale and merging images locally. Server-side rendering is another option if acceptable.

How can I give feedback or contribute?

Open an issue or PR in the GitHub repository (path: tools/tablecopy/). Provide a reproducible example or recording to help debugging.
