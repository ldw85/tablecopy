功能简介

{{TOOL_DESCRIPTION}}

  • 快速、本地运行 - 不上传页面
  • 兼容常见表格与动态可视化
  • 导出为多种格式（CSV, XLSX, PNG, PDF）

工作原理

  1. 脚本在页面上识别表格或可视化 DOM 结构。
  2. 使用 DOM API 提取并序列化数据。
  3. 通过下载或剪贴板 API 将结果传给用户。

如需更改提取逻辑，请编辑 tools/tablecopy/tablecopy_v5.js 中的解析函数（例如 findParentStructure）。

安装与使用

把工具页中的绿色按钮拖到书签栏，打开目标网页后点击书签即可注入脚本并显示导出 UI。

示例

// 在本地打开 index.html 并将按钮拖入书签栏
http://localhost:8000/tools/tablecopy/index.html

常见问题

此工具会上传我的网页内容吗？

不会。默认在本地浏览器执行所有处理，导出结果直接下载到你的设备，除非你主动上传。

打印与截图有什么区别？

打印通常保留文本可搜索性；截图会保留像素级展示但会将内容转为图像。
